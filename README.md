# Snowmanland
Snowmanland is a game built with Unity. It has custom 3D scanned elements, made using Blender, MeshLab and VisualSFM.

The game was made during the "3D Modeling and Reconstruction with Blender & Unity" workshop
2–6 March 2015, Uni­ver­sity of Göttin­gen, Ger­many
Or­gan­ised by the Göttin­gen Cen­tre for Dig­i­tal Hu­man­i­ties (GCDH).
